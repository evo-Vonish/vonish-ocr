# 产品形态全景

VonishOCR 并非单一产品，而是一个覆盖多端、多场景的 OCR 能力矩阵。三种形态共享同一套模型内核与识别管线，仅在部署位置与交互界面上有所区分。

## 本地部署版

本地版是 VonishOCR 的完全体，所有功能无阉割，所有数据不出设备。

| 项目 | 规格 |
|------|------|
| 支持平台 | Windows 10/11、macOS 12+、Linux (AppImage/deb)、Android |
| 模型位置 | `F:\VonishOCR\models\` 或用户指定目录 |
| 服务地址 | `http://localhost:8000` |
| 推理后端 | ONNXRuntime + DirectML / CUDA / CPU |
| 功耗模式 | 野兽 / 均衡 / 节能三档 |
| 离线可用 | 模型下载后，无需任何网络 |

安装完成后，首次启动会自动预加载默认模型（标准档）。你可以通过左侧工具抽屉切换模型档位，或通过后端控制台调整性能模式与并发数。

本地版同时暴露完整的 HTTP API，同一网络内的其他设备（如平板、手机）可访问你的电脑进行识别——你的旧笔记本电脑因此变成了一台本地 OCR 服务器。

## 云端网页版

云端版面向「不想安装任何东西，但此刻需要识别」的用户。

- 浏览器打开即完整可用，无需注册即可试用
- 注册后账号同步配置、API Key、历史记录
- 云端 GPU 集群处理，批量 200 张图片可在数秒内完成
- 文件上传至云端处理后立即删除，不持久化存储

云端版与本地版共享同一套场景分类器、预处理管线与 AI Refiner 配置，识别逻辑一致，结果格式一致。你可以先在云端试用，确认满足需求后再下载本地版长期使用。

## 官方 API

官方 API 面向开发者与系统集成场景。

```
POST https://api.vonishocr.com/v1/ocr
Content-Type: multipart/form-data
```

- 基础 OCR 无限调用，无速率限制
- AI 精修每日附赠 500 次额度
- 超额部分支持自备 API Key 中转（DeepSeek / OpenAI / Qwen）
- 返回格式与本地 API 完全一致，迁移成本为零

## 开发者控制台

无论是本地版还是云端版，内置的开发者控制台提供统一的运维视角：

- **API Key 管理**：创建、轮换、吊销访问密钥
- **用量仪表盘**：按日/按周的调用次数、平均耗时、错误率
- **日志审计**：30 天内的请求记录，支持按状态码与端点筛选

控制台通过顶部状态条的「后端控制台」按钮进入，需要本地管理员权限或云端账号的开发者角色。

---

三种形态之间可以自由迁移：本地版识别的配置导出后，可直接导入云端版；云端版调试好的 AI Refiner 方案，可一键同步至本地。你的设置、你的方案、你的数据，始终归你所有。
