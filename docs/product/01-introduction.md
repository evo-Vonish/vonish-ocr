# 产品介绍

VonishOCR 是一款本地优先的 OCR 识别软件。你的文件不离开设备，模型由你掌控，识别结果归你所有。

在大多数 OCR 工具把图片上传到远端服务器处理的今天，VonishOCR 选择了一条不同的路径：所有推理在本地完成，所有数据停留在你的硬盘上。这不仅是一个技术选择，更是一种主权声明——你的文档、你的笔迹、你的数据，不需要经过任何第三手的视线。

## 核心优势

| 优势 | 说明 |
|------|------|
| 隐私主权 | 文件不离开设备，无需联网即可识别。适合处理合同、病历、试卷等敏感材料 |
| 无限免费基础识别 | 本地模型一次下载，终身免费使用，没有额度焦虑，没有订阅陷阱 |
| 三档模型自动匹配 | 极速 / 标准 / 专业三档模型，系统根据图像质量动态调度，也可手动锁定 |
| Diff 级复核 | 原始 OCR、AI 精修、修改差异三栏并列，红色删除线与绿色新增逐字对照 |
| AI 精修 | 接入 DeepSeek / OpenAI / Qwen 等厂商，对低置信字段自动修正，逐字流式渲染 |
| 完全离线可用 | 下载模型后，无论身处无网络会议室还是保密机房，软件功能完整可用 |

## 三种形态

VonishOCR 以三种形态服务不同场景的用户：

### 本地部署版

算力主权在你手中。支持 Windows、macOS、Linux 与 Android。三档模型按需加载，识别服务运行在 `localhost:8000`，其他应用可通过本地 API 调用。适合对数据安全要求极高的个人用户与企业内网环境。

### 云端网页版

免安装，浏览器即开即用。云端 GPU 加速使处理速度提升 5-10 倍，账号同步配置与历史记录。适合临时使用或设备性能受限的场景。

### 官方 API

7×24 在线的 RESTful 接口，基础识别无限调用，AI 精修每日附赠额度，超额可自备 Key 中转。适合需要将 OCR 能力集成进自有系统的开发者。

## 适用人群

- **财务人员**：发票、收据、对账单的批量识别与结构化导出，直接生成 Excel 或 CSV
- **教师与教研者**：试卷扫描、手写笔记归档、竖排古籍的数字化与 Diff 复核
- **研究者与档案员**：历史文档、信笺、碑刻的 OCR 与校对，低置信字段的人工复核
- **开发者**：通过本地 API 或官方 RESTful 接口，将 VonishOCR 接入自己的工具链与自动化流程
