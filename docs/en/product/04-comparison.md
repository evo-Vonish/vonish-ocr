# Comparison

How VonishOCR differs from the tools you already know.

---

## Vs. Cloud OCR Services

| | VonishOCR | Cloud OCR |
|---|-----------|-----------|
| Files leave device | Never | Always |
| Requires internet | No | Yes |
| Subscription | No | Typically |
| AI polish provider | You choose | Locked in |
| Audit trail | Built in | Hidden |

## Vs. Tesseract

| | VonishOCR | Tesseract |
|---|-----------|-----------|
| Chinese accuracy | High (ONNX models) | Moderate |
| Scene classification | 8-class auto | Manual |
| Preprocessing | 10-step adaptive pipeline | Basic |
| GPU acceleration | DirectML | CPU only |
| AI refinement | Built-in multi-provider | None |

## Vs. PaddleOCR

| | VonishOCR | PaddleOCR |
|---|-----------|-----------|
| Deployment | Single installer | Docker/Python env |
| Model size | 22 MB — 1.87 GB | 1.8 GB+ |
| Desktop UI | Native Tauri + Vue | Web demo only |
| Batch queue | Built-in with WS progress | Script-level |

---

> VonishOCR is not the fastest, the smallest, or the most comprehensive. It is the most honest — and the one where the light stays in your hand.
