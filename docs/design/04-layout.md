# 布局架构

VonishOCR 的主界面以「灯下证据桌」为隐喻，将功能分区与档案室的工作流程对应。

## 五区架构

```
┌─────────────────────────────────────────────────────┐
│  顶部状态条 (48px)                                    │
├──────────┬──────────────────────────────┬─────────────┤
│          │                              │             │
│ 左工具   │      中央证据台面             │  右复核灯   │
│ 抽屉     │      (自适应剩余宽度)          │  (320px)    │
│ (280px)  │                              │             │
│          │                              │             │
│          │                              │             │
├──────────┴──────────────────────────────┴─────────────┤
│  底部读数 (40px)                                      │
└─────────────────────────────────────────────────────┘
```

## 各区域职责

### 左工具抽屉（280px）

- **场景分类**：自动分类标签与手动覆盖选项
- **模型切换**：三档模型选择与加载状态
- **文件列表**：导入的图片缩略图、状态标签、页码
- **批量操作**：全选、反选、删除、导出

抽屉在桌面端常驻显示，在平板端可收缩为图标栏（56px），点击展开。窄屏设备（<768px）抽屉变为底部抽屉（Drawer），手势下滑关闭。

### 中央证据台面（自适应剩余宽度）

- **空状态**：拖拽区提示与快捷操作入口
- **识别中**：扫描线动画 + 预处理日志
- **结果展示**：原图/文本/Diff 的切换视图
- **局部放大**：低置信字段点击后的原图切片弹层

台面是用户注意力最核心的区域。所有动画与状态变化都优先在此呈现。

### 右复核灯（320px）

- **三栏结果**：原始 OCR / AI 精修 / Diff
- **选项卡切换**：180ms 滑动动画 + 掌灯青指示线
- **复核操作**：确认/修改/忽略按钮组
- **导出选项**：格式选择、复制、保存

复核灯在桌面端常驻，在识别完成后自动展开。空状态或识别中时收起，释放台面空间。

### 顶部状态条（48px）

- **后端状态**：就绪/忙碌/异常的指示灯
- **模型档位**：当前模型名称（点击切换）
- **性能模式**：当前模式名称（点击切换）
- **后端控制台入口**：按钮
- **主题切换**：深色/浅色/黑白/高对比度

状态条始终可见，即使在最大化图片预览时也会以半透明背景悬浮显示。

### 底部读数（40px）

- **当前任务状态**：识别中/精修中/完成/异常
- **预处理日志**：当前步骤的滚动文字
- **进度百分比**：批量任务时显示
- **耗时估算**：ETA 时间
- **快捷操作**：复制结果、导出、打开后端控制台

## 四区关系

| 操作 | 左抽屉 | 中央台面 | 右复核灯 | 状态条 | 底部读数 |
|------|--------|---------|---------|--------|---------|
| 导入图片 | 文件列表追加 | 显示缩略图 | 收起 | 任务计数+1 | 等待中 |
| 开始识别 | 状态标签更新 | 扫描线动画 | 等待展开 | 忙碌灯 | 预处理日志滚动 |
| 识别完成 | 状态变完成 | 显示结果提示 | 自动展开 | 就绪灯 | 完成提示 |
| 切换模型 | 档位高亮变化 | 重载提示 | 清空待刷新 | 模型名更新 | 加载进度 |

## 响应式断点

| 断点 | 宽度范围 | 布局变化 |
|------|---------|---------|
| 桌面 | ≥1280px | 五区全部常驻 |
| 大平板 | 1024-1279px | 复核灯可收起为 48px 图标栏 |
| 平板 | 768-1023px | 左抽屉收缩为图标栏，复核灯可覆盖 |
| 窄屏 | <768px | 单栏堆叠，抽屉与复核灯均为底部 Drawer |

---

> 布局设计的核心是「注意力分配」。中央台面占最大空间，因为它是「正在处理的事物」；复核灯占固定窄栏，因为它是「处理结果的审视区」；工具抽屉占固定宽栏，因为它是「处理手段的选择区」。三区宽度比例（280:自适应:320）经过多轮测试，在 13-27 英寸显示器上均能提供舒适的视觉平衡。
